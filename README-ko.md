# Tone Generator

오디오 장비의 테스트나 기타 일정한 오디오 신호가 필요한 상황에서 쓸 수 있는 톤 오디오 생성 프로그램 입니다.
슬라이더를 조절하여 주파수 또는 볼륨 조절이 가능합니다.
